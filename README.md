# gesture-pc-controller
This is a project that aims to reach its highest point of controlling the PC with gestures, starting with gesture typing using the camera.

## Project Procedure
1) We need to create a database for each language (Spanish, English) and populate it with the most common words, their conjugations, and word sequences. To achieve this, we'll use Python code that will run in the background on a PC to track what we type, allowing it to learn. We're also working on creating an InputMethodService that will track what's typed on a mobile phone, because real conversations are the most realistic environment we have for training the AI ​​in the background.

2) Next, we'll create a website that uses the AI ​​and the existing database to provide a near-real-world environment for testing the database.

3) We'll create a website that trains this AI model, indicating when it correctly identifies the desired word and when it doesn't, so it can learn.

# controlador-de-gestos-para-PC
Este es un proyecto que trata de llegar a su ponto mas alto de controlar con gestos el PC empezando por la escritura con gestos usando la camara

## Procedimiento para el proyecto
1) Necesitamos crear una base de datos para cada idioma (español, ingles) y esta llenarla con las palabras más comunes, la conjugación y la sucesión de palabras; para lograr esto usaremos un código de python que se ejecutará en segundo plano en PC para rastrear lo que escribamos, para aprender y estamos trabajando para crear un InputMothodService que rastreara lo que se escriba en el celular, porque las charlas reales es lo más real que tenemos para entrenar en segundo plano la IA que tendrá nuestro proyecto.
2) Después crearemos una página web que use la IA y la base de datos que tenemos, para tener un entorno casi real para probar la base de datos.
3) Crearemos una página web que entrene este modelo de IA indicando cuando acierte a la palabra que queremos y cuando no para que aprenda.
